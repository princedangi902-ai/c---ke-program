## 🛠️ How to Run Programs

### Prerequisites
- Install a C++ compiler (MinGW, GCC, or Clang)
- Install VS Code or any text editor

### Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/princedangi902-ai/c-program-progress.git
   cd c-program-progress
   ```

2. **Navigate to desired folder:**
   ```bash
   cd basics
   ```

3. **Compile and run:**
   ```bash
   # Using g++
   g++ filename.cpp -o filename
   .\filename.exe    # Windows
   ./filename        # Linux/Mac
   ```

4. **Or use VS Code Run Extension:**
   - Install "Code Runner" extension
   - Press `Ctrl + Alt + N` to run

## 🐛 Common Mistakes to Avoid

1. **Data Type Overflow** - Use appropriate data types for large numbers
2. **Off-by-one Errors** - Be careful with loop boundaries
3. **Uninitialized Variables** - Always initialize before use
4. **Integer Division** - Remember: 5/2 = 2 (not 2.5)
5. **Infinite Loops** - Always ensure loop termination condition
